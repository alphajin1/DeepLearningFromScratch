# 이번 장에서 배운 내용
* 추론기반방법은 추측하는 것이 목적이며, 그 부산물로 단어의 분산표현을 얻을 수 있다.
* word2vec은 추론기반방법이며 단순한 2층 신경망이다.
* word2vec은 skip-gram, cbow 모델을 제공한다.
* cbow 모델은 여러 단어(맥락)로부터 하나의 단어(target)를 추측한다.
* 반대로, skip-gram 모델은 하나의 단어(target)로부터 다수의 단어(맥락)을 추측한다.
* word2vec은 가중치를 다시 학습할 수 있으므로, 단어의 분산표현 갱신이나 새로운 단어추가를 효율적으로 수행할 수 있다.
